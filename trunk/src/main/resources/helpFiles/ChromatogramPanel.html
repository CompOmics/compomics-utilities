Chromatogram Panel

  • Zoom In: Left Click and drag in the spectrum to select a time interval to zoom in on.

  • Undo Zoom: Right Click to zooms out to the complete chromatogram.

  • Show Peak Details: Hold the cursor over a peak to display the time value for that peak.

  • Peak Picking: Left Click on a peak to start peak picking. Move the cursor to another peak, to display the time distance between the peaks. Left Click on the second peak to lock the distance and enable peak picking to continue from the last selected peak.

  • Remove Peak Picking: Ctrl + Left Click removes the primary peak picking.

  • Store Peak Picking: Alt + Left Click holds the primary peak picking in a red font and starts a secondary peak picking. Only one peak picking can be stored at the same time.

  • Remove Stored Peak Picking: Ctrl + Alt + Left Click removes the stored pick picking.
