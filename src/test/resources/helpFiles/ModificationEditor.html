Introduction

SearchGUI allows you to inspect implemented modifications as well as manage user modifications in a user friendly way. If you encounter any issues with modifications in SearchGUI, please contact the developers.

Default Modifications

There is along list of default modifications. These cannot be edited. You can however add your own custom modifications (see below). If you need help adding a custom modification, or find an error with one of the default modifications, please contact the developers.

User Modifications

User modifications can be implemented very easily in SearchGUI. Note that user modifications will be available in other instances of SearchGUI and PeptideShaker.
